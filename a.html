Ques.1    A secret code encrypts a message by putting it in an array and reading down the 
          columns (blanks are replaced by '*' and full stops are added to fill up the array).  
          Write a program that encrypts an input string. 
		
            Example 1:
            Input :"i go to market daily to have fast food"
            step 1 :make the string like --> 
					 i*go*t
					 o*mark
					 et*dai
					 ly*to*
					 have*f
					 ast*fo
					 od....
            Output :"ioelhao**tyasdgm**vt.oadte*.*rao*f.tki*fo."
            Example 2:  
            Input: “Lets go to the sandwich shop today” 
            Output: “Lohdsoe*ewhdtt*ioasoscpy**ah*.gtn*t.”
